# To-Do List App (Python)

A simple command-line To-Do application where tasks are saved in a text file.  
You can add tasks, view them, and delete completed ones.

## How to Run
1. Run the script:
2. Choose one of the options:
- 1: View tasks  
- 2: Add a new task  
- 3: Delete a task  
- 4: Exit app  

3. All tasks are saved in `tasks.txt` in the same folder.

## Features
- Tasks are stored permanently in a text file  
- Add, view, and delete tasks  
- Simple user interface  
- Beginner-friendly file handling
